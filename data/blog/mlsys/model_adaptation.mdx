In practice, we cannot assume that the testing data distribution is the same as the training data distribution. Therefore, it is crucial to adapt the model to the new data distribution. In this post, we will discuss recent advances on model adaptation for generative AI.

## Model adaptation

Model adaptation aims to align model behaviors with the target data distribution. Common techniques include fine-tuning, model merging, knowledge editing, and test-time training.
